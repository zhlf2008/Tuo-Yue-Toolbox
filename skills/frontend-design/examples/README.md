# Frontend Design Examples

This directory contains example implementations demonstrating different aesthetic directions.

## How to Use These Examples

1. **Choose an aesthetic** that matches your project requirements
2. **Study the code** to understand the design patterns
3. **Adapt and customize** for your specific use case
4. **Mix elements** from different examples as needed

## Example Categories

### /minimal
Clean, refined designs with generous whitespace and precise details.

### /maximalist
Vibrant, elaborate designs with rich decorations and bold colors.

### /brutalist
Raw, industrial aesthetics with strong contrasts and visible structure.

### /retro-futuristic
Neon, cyberpunk-inspired interfaces with glitch effects and bold contrasts.

### /editorial
Magazine-style layouts with dramatic typography and striking imagery.

### /organic
Natural textures, soft curves, and earth-inspired color palettes.

## Contributing Examples

To add a new example:
1. Create a new directory under `/examples/` for the aesthetic
2. Include a complete, working implementation (HTML + CSS + optional JS)
3. Add a README.md explaining the design choices
4. Demonstrate the aesthetic principles from SKILL.md

## Example Structure

```
/examples/your-aesthetic/
├── index.html          # Complete working example
├── styles.css          # Extractable styles
├── script.js           # Interactive features (optional)
└── README.md           # Design explanation
```
