# Frontend Design Skill

A comprehensive skill for creating distinctive, production-grade frontend interfaces with intentional aesthetics.

## Core Principles

### Design Thinking Framework
Before coding, establish:
- Purpose and audience
- Aesthetic direction (10+ options)
- Technical constraints
- Differentiation strategy

### Anti-Patterns to Avoid
- Generic AI aesthetics
- Overused fonts and colors
- Predictable layouts
- Cookie-cutter designs

## Aesthetic Directions

- **Minimal/Elegant**: Clean, refined, generous whitespace
- **Maximalist**: Vibrant, elaborate, rich decorations
- **Brutalist**: Raw, industrial, strong contrasts
- **Retro-Futuristic**: Neon, cyberpunk, glitch effects
- **Organic/Natural**: Textures, curves, earth tones
- **Luxury/Refined**: Sophisticated, subtle gradients
- **Playful**: Rounded, bright, fun
- **Editorial**: Dramatic layouts, bold typography
- **Art Deco**: Geometric, metallic, elegant
- **Soft/Pastel**: Gentle hues, dreamy atmosphere
- **Industrial**: Functional, monospace, stark

## Usage Guidelines

### For Landing Pages
- Establish clear visual hierarchy
- Use intentional spacing and typography
- Create memorable visual elements
- Ensure mobile-first responsiveness

### For Dashboards
- Implement consistent component patterns
- Use meaningful color coding
- Prioritize information density vs readability
- Include subtle animations for feedback

### For Component Styling
- Design with reusability in mind
- Create consistent interaction patterns
- Ensure accessibility compliance
- Test across different contexts
