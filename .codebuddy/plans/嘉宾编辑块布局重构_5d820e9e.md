## 用户需求

根据提供的布局图修改海报生成器的嘉宾编辑块布局：

- 左上：照片上传区域（较大尺寸）
- 右上：姓名输入框 + 删除按钮（×）
- 右中：头衔输入框
- 底部：简介文本区域（横跨整个宽度）

当前嘉宾块使用左右两栏布局，需要改为网格布局。

## 技术实现方案

### 实现方法

使用 CSS Grid 布局重新设计嘉宾块结构：

- 照片区域：固定宽度，占据左上两行
- 姓名+删除按钮：右上区域
- 头衔输入框：右中区域
- 简介：底部横跨两列

### 关键修改

1. CSS 样式：将 `.guest-block` 改为 Grid 布局
2. HTML 结构：重新组织为网格结构
3. 照片区域尺寸：从 80px 增大到 120px
